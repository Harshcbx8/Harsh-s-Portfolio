CADV — Computer-Aided Diagnosis & Visualization

Description:

CADV is an AI-driven medical imaging system designed to detect cardiac abnormalities from MRI and CT scans using deep learning–based segmentation and visualization. The platform processes volumetric medical data to identify abnormal cardiac structures, generate heatmaps and segmentation masks, and render interactive 3D visualizations. It also provides synchronized multi-view slices (axial, coronal, and sagittal) to help clinicians easily interpret structural abnormalities and improve diagnostic accuracy.

Key Features & Achievements:

  • Developed a deep learning segmentation pipeline using an Attention U-Net architecture with residual gates, trained on the CARE-Cardiac dataset for accurate multi-structure cardiac segmentation.
  • Implemented automated abnormality detection with heatmaps and segmentation masks to highlight pathological regions within MRI/CT scans.
  • Built synchronized multi-axis slice visualization (axial, coronal, sagittal) enabling detailed anatomical inspection and anomaly localization.
  • Created an interactive 3D volumetric renderer to visualize cardiac structures and abnormalities in real time for improved clinical interpretation.
  • Designed a FastAPI inference backend and Streamlit-based visualization dashboards to deliver GPU-accelerated predictions and interactive diagnostic insights.
  • Improved model explainability by integrating attention-based visual cues, enhancing transparency and practitioner trust in automated diagnosis.
